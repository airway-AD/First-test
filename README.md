# To-Dir

To-Dir est une todo-list moderne (HTML/CSS/JS) orientée **priorisation intelligente** et **focus**.

## Benchmark produit (best practices intégrées)
Inspiré des patterns populaires observés sur Todoist, TickTick, Things et Notion:
- capture rapide + parsing intelligent en langage naturel (`#tag`, `@projet`, `demain`, `p1`, `45m`)
- tri intelligent via score (priorité + urgence + durée + deep work)
- filtres combinés (statut, priorité, recherche)
- rappels d'échéance (aujourd'hui/demain/en retard)
- tâches récurrentes quotidiennes
- mesure de productivité (focus score, terminé aujourd'hui, retard)
- design sombre flat noir/vert type Spotify

## Fonctionnalités
- création de tâches enrichies (projet, tags, priorité, durée, deadline, deep work)
- recommandation automatique de la prochaine meilleure action
- tri smart / échéance / priorité / création
- stockage local (`localStorage`)
- suppression individuelle et nettoyage des tâches terminées

## Lancer
Ouvre simplement `index.html` dans le navigateur.
